HTML 5

HTML5 es la quinta revisión del lenguaje de marcado fundamental de la World Wide Web, HTML. Define una estructura para documentos web y especifica cómo organizar el contenido de una página web. Entre sus novedades encontramos:

  • Nuevos elementos semánticos como <header>, <article> y <footer> que permiten a los navegadores comprender mejor el contenido de la página.
  • Soporte multimedia nativo para audio y video, sin necesidad de plugins adicionales.
  • APIs para acceder a funciones del dispositivo, como la cámara o la geolocalización.
  • Mayor flexibilidad en el diseño y la maquetación de páginas web.
